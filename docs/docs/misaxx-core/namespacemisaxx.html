◆ misa_work_subtree_status

enum misaxx::misa_work_subtree_status : int

Status of the node's subtree

Enumerator
incomplete 

Not all children are instantiated. The runtime must keep track of this node and update it constantly until the node reports completion

complete 

All children are instantiated. The runtime is not required anymore to check for changes in children

◆ misa_worker_status

enum misaxx::misa_worker_status : int
strong

Status of a node

Enumerator
undone 

Work is not done

working 

The task is currently working.

done 

The work is done

queued_repeat 

The worker rejected the work (e.g. via worker::input<>::exists). Work must be retried later

waiting 

The worker is waiting for sub-workers
